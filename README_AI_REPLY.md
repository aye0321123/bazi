## 🔍 问题说明

### 核心问题
当通过 API 向 BaziAI 发送消息后，AI 不会自动生成回复。

### 测试数据
```
总消息数: 24 条
用户消息: 23 条
AI 回复: 1 条
回复率: 4.3%
```

### 原因分析
1. **异步处理**: AI 在后台异步生成回复
2. **触发机制**: 可能需要用户访问官网页面才会触发
3. **API 限制**: 纯 API 调用可能无法完全触发 AI 逻辑

---

## ✅ 解决方案

### 方案概述
实现了**双模式发送系统**，让用户可以根据需求选择：

#### 模式 1: 快速发送 📤
- 立即发送，不等待回复
- 响应时间 < 1 秒
- 需要手动刷新查看回复

#### 模式 2: 发送并等待回复 ⏳ (推荐)
- 发送后自动轮询等待
- 最多等待 60 秒
- 自动显示回复或超时提示

### 技术实现

```
┌─────────────┐
│  用户输入   │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  发送消息   │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  轮询检查   │◄─── 每 3 秒检查一次
│  (最多60秒) │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ 收到回复？  │
└──────┬──────┘
       │
   ┌───┴───┐
   │       │
  是      否
   │       │
   ▼       ▼
显示回复  超时提示
```

```
1. 打开 http://127.0.0.1:5000
2. 输入 Session ID 和 Cookie
3. 点击"登录天机阁"
```

## 📡 API 文档

### 端点 1: 快速发送

```http
POST /api/send
Content-Type: application/json

{
  "content": "今天运势如何？"
}
```

**响应**:
```json
{
  "success": true,
  "data": {
    "id": "message-id",
    "role": "user",
    "content": "今天运势如何？",
    "created_at": "2026-04-18T10:00:00Z"
  }
}
```

### 端点 2: 发送并等待回复

```http
POST /api/send-and-wait
Content-Type: application/json

{
  "content": "今天运势如何？",
  "max_wait": 60,
  "check_interval": 3
}
```

**参数说明**:
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| content | string | 是 | - | 消息内容 |
| max_wait | number | 否 | 60 | 最大等待时间（秒） |
| check_interval | number | 否 | 3 | 检查间隔（秒） |

**成功响应**:
```json
{
  "success": true,
  "user_message": {
    "id": "user-msg-id",
    "role": "user",
    "content": "今天运势如何？"
  },
  "ai_reply": {
    "id": "ai-msg-id",
    "role": "assistant",
    "content": "根据您的八字分析，今天财运...",
    "created_at": "2026-04-18T10:00:15Z"
  },
  "wait_time": 15,
  "attempts": 5
}
```

**超时响应**:
```json
{
  "success": false,
  "error": "等待超时，AI 未回复",
  "user_message": {...},
  "wait_time": 60,
  "attempts": 20,
  "suggestion": "请到 BaziAI 官网查看回复"
}
```

### 端点 3: 获取消息列表

```http
GET /api/messages
```

**响应**:

## ❓ 常见问题

### Q1: 为什么 AI 不回复？

**A**: 可能的原因：
1. AI 需要时间生成回复（通常 5-60 秒）
2. BaziAI 可能需要用户访问官网才触发回复
3. Cookie 可能已过期
4. 网络连接问题

**解决方法**:
- 使用"发送并等待回复"模式
- 增加等待时间
- 访问 BaziAI 官网查看
- 更新 Cookie

### Q2: 等待时间太长怎么办？

**A**: 
- 默认等待 60 秒，可以调整 `max_wait` 参数
- 使用"快速发送"模式，稍后手动刷新
- 开启调试模式查看详细日志

### Q3: 如何提高回复率？

**A**: 
1. 发送消息后访问官网页面
2. 使用浏览器自动化（Selenium）
3. 增加轮询时间和频率
4. 确保 Cookie 有效

### Q4: Cookie 多久过期？

**A**: 
- 通常 30 天左右
- 查看 `bazi_credentials.json` 中的 `expiry` 字段
- 过期前需要重新获取

### Q5: 如何调试问题？

**A**: 
1. 开启前端调试模式（勾选"调试模式"）
2. 查看浏览器控制台日志
3. 运行 `python check_ai_reply.py` 检查状态
4. 查看 Flask 后端日志

### Q6: 可以批量发送消息吗？

**A**: 
- 可以，但不推荐同时使用"等待回复"模式
- 建议使用"快速发送"模式批量发送
- 然后统一刷新查看回复

### Q7: 如何保存聊天记录？

**A**: 
- 点击"导出"按钮下载 JSON 文件
- 使用 `python get_chat_history.py` 导出
- 聊天记录自动保存在 BaziAI 服务器
