# 🔮 天机阁使用说明

## 📋 功能概述

天机阁是一个基于 Flask 的 Web 应用，可以通过友好的界面调用 BaziAI 的 API。

### ✨ 新增功能

1. **创建新对话** - 一键创建新的聊天会话
2. **发送消息** - 在当前会话中发送消息
3. **刷新消息** - 获取最新的对话内容
4. **导出聊天** - 将对话记录导出为 JSON 文件

---

## 🚀 快速开始

### 步骤 1: 启动应用

```bash
python app.py
```

应用将在 `http://127.0.0.1:5000` 启动

### 步骤 2: 打开浏览器

访问：http://127.0.0.1:5000

### 步骤 3: 登录配置

#### 方式 A: 使用现有会话（推荐新手）

1. **获取 Cookie**
   - 打开 https://www.bazi-ai.com
   - 登录你的账号
   - 按 F12 打开开发者工具
   - 切换到 **Application** 标签
   - 左侧选择 **Cookies** → **https://www.bazi-ai.com**
   - 复制所有 Cookie（格式：`name1=value1; name2=value2; ...`）

2. **获取 Session ID**
   - 在 BaziAI 网站上打开任意对话
   - 从地址栏复制 Session ID
   - 格式：`https://www.bazi-ai.com/zh/chat/[这里是Session ID]`

3. **在天机阁中登录**
   - 粘贴 Session ID
   - 粘贴 Cookie
   - 点击"登录天机阁"

#### 方式 B: 从凭证文件读取（推荐高级用户）

如果你已经运行过 `get_cookie_v2.py`，可以直接使用保存的凭证：

```python
import json

# 读取凭证
with open('bazi_credentials.json', 'r', encoding='utf-8') as f:
    creds = json.load(f)

print("Cookie:", creds['cookie'])
print("Session ID: 26a7d080-283c-43c9-a741-23d8dfcb8512")  # 你的会话ID
```

## 💡 使用功能

### 1. 创建新对话

点击 **"✨ 新对话"** 按钮：

- **选项 1**: 输入第一条消息，创建对话并发送
- **选项 2**: 留空，只创建新对话

创建成功后：
- 会显示新的会话 ID
- 可以选择在新标签页中打开
- 自动切换到新会话

### 2. 发送消息

在输入框中输入消息，点击 **"📤 发送"**

- 支持多行输入（Shift + Enter 换行）
- 按 Enter 直接发送
- 消息会立即显示在对话框中

### 3. 查看 AI 回复

**重要提示**：AI 回复需要在 BaziAI 官网上才能触发！

#### 方法 1: 在官网查看（推荐）

1. 创建新对话后，点击"在新标签页中打开"
2. 或手动访问：`https://www.bazi-ai.com/zh/chat/[会话ID]`
3. 等待 5-10 秒，AI 会自动回复

#### 方法 2: 在天机阁中查看

1. 在官网上等待 AI 回复完成
2. 回到天机阁
3. 点击 **"🔄 刷新"** 按钮
4. AI 的回复会显示在对话框中

### 4. 导出聊天记录

点击 **"📥 导出"** 按钮：

- 自动下载 JSON 文件
- 文件名格式：`bazi_chat_20260417_143025.json`
- 包含完整的对话历史

## 🔧 调试模式

如果遇到问题，可以启用调试模式：

1. 在登录页面勾选 **"🔧 调试模式"**
2. 底部会显示调试日志
3. 可以查看详细的请求和响应信息

## 📡 API 端点说明

### 1. POST `/api/login`
登录并验证凭证

**请求体**：
```json
{
  "session_id": "会话ID",
  "cookie": "完整的Cookie字符串"
}
```

**响应**：
```json
{
  "success": true,
  "message": "登录成功"
}
```

---

### 2. POST `/api/create-session`
创建新的聊天会话

**响应**：
```json
{
  "success": true,
  "session_id": "新会话ID",
  "data": {
    "uuid": "新会话ID",
    "created_at": "2026-04-17T06:41:59.836452+00:00",
    "model": "r1"
  }
}
```

---

### 3. POST `/api/create-and-send`
创建新对话并发送消息

**请求体**：
```json
{
  "content": "你好，请用一句话介绍你自己"
}
```

**响应**：
```json
{
  "success": true,
  "session_id": "新会话ID",
  "message_data": {
    "id": "消息ID",
    "content": "你好，请用一句话介绍你自己",
    "created_at": "2026-04-17T06:42:00.123456+00:00"
  },
  "chat_url": "https://www.bazi-ai.com/zh/chat/新会话ID"
}
```

---

### 4. POST `/api/send`
在当前会话中发送消息

**请求体**：
```json
{
  "content": "请帮我分析一下今天的运势"
}
```

**响应**：
```json
{
  "success": true,
  "data": {
    "id": "消息ID",
    "content": "请帮我分析一下今天的运势",
    "created_at": "2026-04-17T06:42:00.123456+00:00"
  }
}
```

---

### 5. GET `/api/messages`
获取当前会话的所有消息

**响应**：
```json
{
  "success": true,
  "data": [
    {
      "id": "消息ID",
      "role": "user",
      "content": "消息内容",
      "created_at": "2026-04-17T06:42:00.123456+00:00"
    },
    {
      "id": "消息ID",
      "role": "assistant",
      "content": "AI回复内容",
      "created_at": "2026-04-17T06:42:05.123456+00:00"
    }
  ]
}
```

---

### 6. GET `/api/export`
导出聊天记录

**响应**：
```json
{
  "success": true,
  "filename": "bazi_chat_20260417_143025.json",
  "data": [...]
}
```

---

### 7. POST `/api/logout`
登出并清除会话

**响应**：
```json
{
  "success": true,
  "message": "已登出"
}
```

```
1. 登录天机阁（使用现有 Session ID 和 Cookie）
   ↓
2. 发送消息
   ↓
3. 在 BaziAI 官网查看 AI 回复
   ↓
4. 回到天机阁，点击"刷新"查看完整对话
```

### 场景 2: 创建新对话

```
1. 登录天机阁（只需要 Cookie，不需要 Session ID）
   ↓
2. 点击"✨ 新对话"
   ↓
3. 输入第一条消息（或留空）
   ↓
4. 系统自动创建新会话并发送消息
   ↓
5. 在新标签页中打开对话
   ↓
6. 在 BaziAI 官网查看 AI 回复
```

## ⚠️ 注意事项

### 1. Cookie 安全
- 不要分享你的 Cookie
- Cookie 通常 30 天过期
- 过期后需要重新获取

### 2. AI 回复机制
- **重要**：AI 回复只能在 BaziAI 官网上触发
- 天机阁的 API 只能发送消息和获取历史
- 发送消息后，必须在官网上等待 AI 回复

### 3. 会话管理
- 每次创建新对话会生成新的 Session ID
- 旧的 Session ID 仍然有效，可以继续使用
- 可以在多个会话之间切换（需要重新登录）

### 4. 网络问题
- 如果遇到 SSL 错误，检查网络连接
- 如果请求超时，增加等待时间
- 启用调试模式查看详细错误信息

## 🆘 常见问题

### Q: 为什么发送消息后没有 AI 回复？
A: AI 回复需要在 BaziAI 官网上触发。发送消息后，访问官网对应的对话页面，等待 5-10 秒即可看到回复。

### Q: 如何在天机阁中查看 AI 回复？
A: 在官网上等待 AI 回复完成后，回到天机阁，点击"🔄 刷新"按钮。

### Q: Cookie 过期了怎么办？
A: 运行 `python get_cookie_v2.py` 重新获取 Cookie，或手动从浏览器复制。

### Q: 可以同时使用多个会话吗？
A: 可以，但需要每次切换时重新登录（输入不同的 Session ID）。

### Q: 创建新对话后，旧对话还能用吗？
A: 可以！旧的 Session ID 仍然有效，只需重新登录即可切换回去。
