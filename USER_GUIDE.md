## 🚀 快速开始

### 第一步：访问应用

打开浏览器，访问管理员提供的网址，例如：
- `http://your-domain.com`
- `http://123.456.789.0:5000`

### 第二步：准备登录信息

您需要两个信息才能使用：

#### 1. Session ID（会话ID）
- 登录 [BaziAI 官网](https://www.bazi-ai.com)
- 从浏览器地址栏复制会话ID
- 格式示例：`26a7d080-283c-43c9-a741-23d8dfcb8512`

**完整地址示例**：
```
https://www.bazi-ai.com/zh/chat/26a7d080-283c-43c9-a741-23d8dfcb8512
                                    ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                    这部分就是 Session ID
```

#### 2. Cookie（身份凭证）
这是您的登录凭证，获取步骤：

1. 在 BaziAI 网站上按 `F12` 键打开开发者工具
2. 点击顶部的 **Network（网络）** 标签
3. 点击 **Fetch/XHR** 过滤器
4. 在网站上发送一条测试消息
5. 在列表中找到并点击 `messages` 请求
6. 在右侧找到 **Headers（标头）** 标签
7. 向下滚动找到 **Request Headers（请求标头）**
8. 找到 **Cookie** 字段
9. 复制完整的 Cookie 值（很长的一串文本）

**Cookie 示例**（已脱敏）：
```
NEXT_LOCALE=zh; _ga=GA1.1.xxx; __Secure-authjs.session-token=eyJhbGci...
```

### 第三步：登录

1. 在登录页面，将 **Session ID** 粘贴到第一个输入框
2. 将 **Cookie** 粘贴到第二个输入框（可以多行）
3. 点击 **登录** 按钮
4. 等待验证（通常需要 2-5 秒）

✅ 如果看到"登录成功"，说明配置正确！

## 💬 使用聊天功能

### 发送消息

1. 在底部的输入框中输入您的问题
2. 点击 **发送** 按钮（或按 `Enter` 键）
3. 等待 AI 回复（通常需要几秒钟）
4. 点击 **刷新** 按钮查看 AI 的回复

### 常见问题示例

您可以问这些问题：

```
请帮我分析一下2026年的运势

我适合什么职业？

我的财运怎么样？

请根据我的八字帮我制定一份人生计划

预测我在2026年的关键词

我的MBTI人格是INTJ，请给我一些职业建议
```

### 查看历史消息

- 所有消息会自动显示在聊天窗口中
- 点击 **刷新** 按钮可以重新加载消息
- 向上滚动可以查看更早的消息

1. 点击右上角的 **导出** 按钮
2. 浏览器会自动下载一个 JSON 文件

### 查看导出的文件

JSON 文件可以用以下方式打开：
- 文本编辑器（记事本、VS Code 等）
- 在线 JSON 查看器
- 导入到其他应用程序

## 🔐 安全提示

### ⚠️ 重要：保护您的 Cookie

Cookie 是您的登录凭证，相当于密码：

1. **不要分享** Cookie 给其他人
2. **定期更换** - Cookie 会过期，需要重新获取
3. **安全存储** - 如果保存 Cookie，请使用密码管理器
4. **退出登录** - 使用完毕后点击"登出"按钮

### Cookie 过期怎么办？

如果看到"认证失败"或"401 错误"：

1. 重新登录 BaziAI 官网
2. 按照上述步骤重新获取 Cookie
3. 在应用中重新登录

## ❓ 常见问题

### Q1: 登录失败怎么办？

**可能原因**：
- Session ID 不正确
- Cookie 已过期
- Cookie 复制不完整

**解决方法**：
1. 检查 Session ID 格式是否正确
2. 重新获取 Cookie
3. 确保复制了完整的 Cookie（包括开头和结尾）

### Q2: 发送消息后没有回复？

**解决方法**：
1. 等待 3-5 秒
2. 点击"刷新"按钮
3. 如果还是没有，检查网络连接
4. 尝试重新登录

### Q3: 显示"未登录"？

**原因**：Session 过期

**解决方法**：
1. 刷新页面
2. 重新输入 Session ID 和 Cookie
3. 点击登录

### Q4: Cookie 在哪里找？

**详细步骤**：
1. 打开 BaziAI 网站
2. 按 `F12` 键
3. 点击 `Network` 标签
4. 点击 `Fetch/XHR` 过滤器
5. 在网站上发送一条消息
6. 点击 `messages` 请求
7. 在右侧找到 `Headers` → `Request Headers` → `Cookie`

### Q5: 可以在手机上使用吗？

**可以！** 但获取 Cookie 比较困难：

**推荐方法**：
1. 在电脑上获取 Session ID 和 Cookie
2. 通过邮件或笔记应用发送到手机
3. 在手机浏览器中打开应用
4. 粘贴 Session ID 和 Cookie

### Q6: 多个人可以同时使用吗？

**可以！** 每个人使用自己的：
- Session ID（从自己的 BaziAI 账号获取）
- Cookie（从自己的浏览器获取）

**注意**：不要共享 Cookie，每个人应该使用自己的账号。
