# 🚀 天机阁 - 快速开始

## 📋 3 步开始使用

### 步骤 1: 启动应用

```bash
python app.py
```

✅ 看到这个提示说明启动成功：
```
* Running on http://127.0.0.1:5000
```

---

### 步骤 2: 打开浏览器

访问：**http://127.0.0.1:5000**

你会看到"天机阁"的登录页面

---

### 步骤 3: 登录

#### 选项 A: 使用保存的凭证（推荐）

如果你已经运行过 `get_cookie_v2.py`：

1. 打开 `bazi_credentials.json` 文件
2. 复制 `cookie` 的值
3. 粘贴到天机阁的 Cookie 输入框
4. Session ID 可以留空或填写任意值
5. 点击"登录天机阁"

#### 选项 B: 手动获取 Cookie

1. 打开 https://www.bazi-ai.com
2. 登录你的账号
3. 按 **F12** 打开开发者工具
4. 切换到 **Application** 标签
5. 左侧选择 **Cookies** → **https://www.bazi-ai.com**
6. 复制所有 Cookie（格式：`name1=value1; name2=value2; ...`）
7. 粘贴到天机阁
8. 点击"登录天机阁"

---

## 🎯 开始使用

### 功能 1: 创建新对话

1. 登录后，点击 **"✨ 新对话"** 按钮
2. 输入你的第一条消息，例如：
   ```
   你好，请用一句话介绍你自己
   ```
3. 点击"确定"
4. 系统会提示是否在新标签页中打开
5. 点击"确定"打开对话页面
6. 等待 5-10 秒，AI 会自动回复

### 功能 2: 发送消息

1. 在输入框中输入消息
2. 点击 **"📤 发送"** 或按 **Enter**
3. 消息会立即显示在对话框中
4. 在 BaziAI 官网查看 AI 回复
5. 回到天机阁，点击 **"🔄 刷新"** 查看完整对话

### 功能 3: 导出聊天

1. 点击 **"📥 导出"** 按钮
2. 自动下载 JSON 文件
3. 文件包含完整的对话历史

## 💡 快速提示

### ✅ 推荐工作流程

```
1. 在天机阁中创建新对话并发送消息
   ↓
2. 在新标签页中打开对话（自动跳转）
   ↓
3. 在 BaziAI 官网等待 AI 回复（5-10秒）
   ↓
4. 回到天机阁，点击"刷新"查看完整对话
```

### ⚠️ 重要提示

1. **AI 回复机制**
   - AI 回复只能在 BaziAI 官网上触发
   - 天机阁的 API 只能发送消息和获取历史
   - 发送消息后，必须在官网上等待 AI 回复

2. **Cookie 安全**
   - 不要分享你的 Cookie
   - Cookie 通常 30 天过期
   - 过期后需要重新获取

3. **调试模式**
   - 遇到问题时，勾选"🔧 调试模式"
   - 可以查看详细的请求和响应日志

## 🔧 故障排除

### 问题 1: 登录失败

**错误信息**：`连接失败: SSL 连接错误`

**解决方法**：
```bash
# 安装 brotli 包
pip install brotli
```

### 问题 2: Cookie 过期

**错误信息**：`401 Unauthorized`

**解决方法**：
```bash
# 重新获取 Cookie
python get_cookie_v2.py
```

### 问题 3: 没有 AI 回复

**原因**：AI 回复需要在官网上触发

**解决方法**：
1. 复制对话链接
2. 在浏览器中打开
3. 等待 5-10 秒
4. 回到天机阁刷新

### 问题 4: 应用无法启动

**错误信息**：`ModuleNotFoundError: No module named 'flask'`

**解决方法**：
```bash
# 安装依赖
pip install flask requests brotli
```
