## 🔧 常见问题

### Q1: 部署失败怎么办？

**检查以下几点：**
- [ ] GitHub 仓库是否可访问
- [ ] render_cloud.yaml 配置是否正确
- [ ] Python 版本是否为 3.12.0
- [ ] 依赖文件是否完整

**解决方案：**
1. 查看 Render 部署日志
2. 检查错误信息
3. 重新部署

---

### Q2: 部署成功但无法访问？

**可能原因：**
- [ ] 服务正在启动（等待 1-2 分钟）
- [ ] API 地址错误
- [ ] 网络连接问题

**解决方案：**
1. 等待服务完全启动
2. 检查 API 地址是否正确
3. 在 Render Dashboard 查看服务状态

---

### Q3: API 返回错误？

**可能原因：**
- [ ] Cookie 已过期
- [ ] 请求格式错误
- [ ] BaziAI 服务不可用

**解决方案：**
1. 更新 Cookie
2. 检查请求头和参数
3. 查看 API 文档
