## ⚠️ 注意事项

### 1. 等待时间
- 正常等待时间：10-15 秒
- 如果超过 30 秒未回复，可能需要刷新页面

### 2. Cookie 有效期
- Cookie 有效期约 30 天
- 过期后需要重新获取

### 3. 网络连接
- 确保网络连接稳定
- Selenium 需要访问 BaziAI 官网

### 4. Chrome 浏览器
- 系统需要安装 Chrome 浏览器
- webdriver-manager 会自动下载匹配的 ChromeDriver

## 🔧 故障排查

### 问题 1: "Selenium 未安装"
**解决方法**:
```bash
pip install selenium webdriver-manager
```

### 问题 2: 等待超时，未收到回复
**可能原因**:
- AI 生成时间较长
- 网络连接问题
- Cookie 过期

**解决方法**:
1. 尝试使用"🌐 在官网查看"按钮
2. 检查 Cookie 是否有效
3. 查看 Flask 控制台日志

### 问题 3: ChromeDriver 错误
**解决方法**:
- webdriver-manager 会自动处理
- 如果仍有问题，更新 Chrome 到最新版本

### 问题 4: 页面无法访问
**解决方法**:
1. 确认 Flask 正在运行
2. 检查端口 5000 是否被占用
3. 尝试重启 Flask
