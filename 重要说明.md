---

## 💡 原因分析

### BaziAI 的工作机制

BaziAI 使用了**实时通信技术**来推送 AI 回复：

1. **WebSocket** - 用于实时双向通信
2. **Server-Sent Events (SSE)** - 用于服务器推送
3. **前端触发机制** - AI 回复需要在网页上才能触发

### 为什么 API 调用没有回复？

```
用户在网页发送消息
    ↓
前端建立 WebSocket 连接
    ↓
后端触发 AI 生成
    ↓
通过 WebSocket 推送回复
    ↓
前端显示回复
```

**但是通过 API 直接调用**：
```
API 发送消息
    ↓
消息保存到数据库
    ↓
❌ 没有 WebSocket 连接
    ↓
❌ AI 不会被触发
    ↓
❌ 没有回复
```

## 🎯 API 的实际用途

虽然 API 不能触发 AI 回复，但它仍然非常有用：

### ✅ 可以做的事情

1. **获取聊天记录**
   ```python
   messages = client.get_messages()
   ```

2. **导出聊天数据**
   ```python
   client.save_messages_to_file(messages)
   ```

3. **分析聊天内容**
   ```python
   # 统计消息数量
   user_count = len([m for m in messages if m['role'] == 'user'])
   ai_count = len([m for m in messages if m['role'] == 'assistant'])
   
   # 提取关键词
   # 生成报告
   # 等等...
   ```

4. **监控新消息**
   ```python
   # 定期检查是否有新的 AI 回复
   # 发送通知
   # 自动保存
   ```

5. **批量处理**
   ```python

   # 数据分析
   # 备份
   ```

### ❌ 不能做的事情

## 🔧 实用脚本

### 1. 监控新消息

```python
# monitor_messages.py
import time
from bazi_api import BaziAI
import json

with open('bazi_credentials.json', 'r') as f:
    cookie = json.load(f)['cookie']

client = BaziAI("26a7d080-283c-43c9-a741-23d8dfcb8512", cookie)

last_count = 0

while True:
    messages = client.get_messages()
    current_count = len(messages)
    
    if current_count > last_count:
        print(f"🔔 发现 {current_count - last_count} 条新消息！")
        new_messages = messages[last_count:]
        for msg in new_messages:
            role = "👤" if msg['role'] == 'user' else "🤖"
            print(f"{role} {msg['content'][:50]}...")
        last_count = current_count
    
    time.sleep(10)
```

### 2. 自动导出

```python
# auto_export.py
import schedule
import time
from bazi_api import BaziAI
import json

with open('bazi_credentials.json', 'r') as f:
    cookie = json.load(f)['cookie']

client = BaziAI("26a7d080-283c-43c9-a741-23d8dfcb8512", cookie)

def export_daily():
    messages = client.get_messages()
    client.save_messages_to_file(messages)
    print(f"✅ 已导出 {len(messages)} 条消息")

# 每天晚上 10 点导出
schedule.every().day.at("22:00").do(export_daily)

while True:
    schedule.run_pending()
    time.sleep(60)
```
