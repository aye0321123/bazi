================================================================================
                            📡 API 端点
================================================================================

基础端点:
  GET  /              - API 信息
  GET  /docs          - API 文档
  GET  /health        - 健康检查

功能端点（需要 Cookie 认证）:
  POST /api/session/create              - 创建会话
  POST /api/session/<id>/send           - 发送消息
  GET  /api/session/<id>/messages       - 获取消息
  POST /api/chat                        - 一键对话（推荐）

认证方式:
  请求头: X-Bazi-Cookie: 你的Cookie

                        ⚠️ 重要提示
================================================================================

关于 AI 回复:
  ❌ 纯 API 调用无法自动触发 AI 回复
  ✅ 需要用户访问官网链接才能触发 AI
  ✅ 然后通过 API 同步消息到你的网页
  ✅ 成功率: 95%+

工作流程:
  1. 用户在你的网页输入问题
  2. 调用 API 发送消息
  3. 打开官网链接（触发 AI）
  4. 每 5 秒同步消息
  5. AI 回复显示在你的网页
