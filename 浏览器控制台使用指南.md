# 🌐 在浏览器中直接调用 BaziAI API

## 📋 使用步骤

### 步骤 1: 打开网页
访问：https://www.bazi-ai.com/zh/chat

### 步骤 2: 打开开发者工具
- **Windows**: 按 `F12` 或 `Ctrl + Shift + I`
- **Mac**: 按 `Cmd + Option + I`

### 步骤 3: 切换到控制台
点击 **Console** (控制台) 标签

### 步骤 4: 复制粘贴代码
复制下面的代码，粘贴到控制台，然后按回车

## 🚀 快速使用代码

### 方式 1: 创建新对话并发送消息（推荐）

```javascript
// 复制这段代码到浏览器控制台
(async function() {
    // 创建新对话
    const response1 = await fetch('https://www.bazi-ai.com/api/chat-session', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({})
    });
    const result1 = await response1.json();
    const sessionId = result1.data.uuid;
    console.log('✅ 新对话创建成功！会话 ID:', sessionId);
    
    // 发送消息
    const message = '你好，请用一句话介绍你自己';  // 👈 修改这里的消息内容
    const response2 = await fetch(`https://www.bazi-ai.com/api/chat-session/${sessionId}/messages`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            session_id: sessionId,
            role: 'user',
            content: message,
            id: ''
        })
    });
    const result2 = await response2.json();
    console.log('✅ 消息已发送！消息 ID:', result2.id);
    
    // 跳转到新对话
    const chatUrl = `https://www.bazi-ai.com/zh/chat/${sessionId}`;
    console.log('🌐 对话链接:', chatUrl);
    console.log('⏳ 2秒后自动跳转...');
    setTimeout(() => window.location.href = chatUrl, 2000);
})();
```

### 方式 2: 在当前对话中发送消息

如果你已经打开了一个对话页面，可以直接发送消息：

```javascript
// 复制这段代码到浏览器控制台
(async function() {
    // 从 URL 获取当前会话 ID
    const sessionId = window.location.pathname.split('/').pop();
    console.log('📍 当前会话 ID:', sessionId);
    
    // 发送消息
    const message = '请帮我分析一下今天的运势';  // 👈 修改这里的消息内容
    const response = await fetch(`https://www.bazi-ai.com/api/chat-session/${sessionId}/messages`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            session_id: sessionId,
            role: 'user',
            content: message,
            id: ''
        })
    });
    const result = await response.json();
    console.log('✅ 消息已发送！消息 ID:', result.id);
    console.log('💡 等待几秒钟，AI 回复会自动显示在页面上');
})();
```

### 方式 3: 获取当前对话的所有消息

```javascript
// 复制这段代码到浏览器控制台
(async function() {
    // 从 URL 获取当前会话 ID
    const sessionId = window.location.pathname.split('/').pop();
    
    // 获取消息列表
    const response = await fetch(`https://www.bazi-ai.com/api/chat-session/${sessionId}/messages`);
    const messages = await response.json();
    
    console.log(`✅ 获取到 ${messages.length} 条消息`);
    messages.forEach((msg, i) => {
        const role = msg.role === 'user' ? '👤 用户' : '🤖 AI';
        console.log(`\n${i+1}. ${role} [${msg.created_at}]`);
        console.log(`   ${msg.content.substring(0, 100)}...`);
    });
})();
```

## 📝 注意事项

1. **Cookie 自动携带**
   - 在浏览器控制台中运行代码，Cookie 会自动携带
   - 不需要手动设置 Cookie

2. **AI 回复**
   - 消息发送后，AI 回复会在页面上自动显示
   - 通常需要等待 3-10 秒

3. **跨域问题**
   - 必须在 `bazi-ai.com` 域名下运行
   - 不能在其他网站的控制台运行

4. **刷新页面**
   - 刷新页面后需要重新粘贴代码
   - 或者将代码保存为书签（Bookmarklet）

## 🆘 常见问题

### Q: 代码运行后没有反应？
A: 检查是否在 `bazi-ai.com` 域名下，按 F12 查看控制台是否有错误信息

### Q: 提示 "Uncaught SyntaxError"？
A: 确保复制了完整的代码，包括开头的 `(async function() {` 和结尾的 `})();`

### Q: AI 没有回复？
A: 等待 5-10 秒，AI 回复需要时间。如果还是没有，刷新页面查看

### Q: 如何修改发送的消息？
A: 找到代码中的 `const message = '...'` 这一行，修改引号内的内容
