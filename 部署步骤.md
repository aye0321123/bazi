### 方法2: Python 测试

**创建测试脚本 `test_render.py`：**

```python
import requests
import json

# 替换为你的 Render URL
API_URL = "https://你的地址.onrender.com"

# 从文件读取 Cookie
with open('bazi_credentials.json', 'r', encoding='utf-8') as f:
    COOKIE = json.load(f)['cookie']

# 测试一键对话
response = requests.post(
    f"{API_URL}/api/chat",
    headers={
        "X-Bazi-Cookie": COOKIE,
        "Content-Type": "application/json"
    },
    json={"content": "今天运势如何？"}
)

result = response.json()

if result['success']:
    print(f"✅ 成功！")
    print(f"Session ID: {result['session_id']}")
    print(f"官网链接: {result['chat_url']}")
else:
    print(f"❌ 失败: {result['error']}")
```

**运行测试：**

## 📡 在网页中使用

### 完整示例（复制即用）

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>BaziAI 测试</title>
</head>
<body>
    <h1>🔮 BaziAI Cloud API</h1>
    
    <input type="text" id="question" placeholder="输入问题" value="今天运势如何？">
    <button onclick="ask()">发送</button>
    
    <div id="result"></div>

    <script>
        const API_URL = "https://你的地址.onrender.com";
        const COOKIE = "你的Cookie";
        
        async function ask() {
            const question = document.getElementById('question').value;
            const resultDiv = document.getElementById('result');
            
            resultDiv.innerHTML = '⏳ 发送中...';
            
            const response = await fetch(`${API_URL}/api/chat`, {
                method: 'POST',
                headers: {
                    'X-Bazi-Cookie': COOKIE,
                    'Content-Type': 'application/json'
                },
                body: JSON.stringify({ content: question })
            });
            
            const result = await response.json();
            
            if (result.success) {
                resultDiv.innerHTML = `
                    ✅ 发送成功！<br>
                    Session ID: ${result.session_id}<br>
                    <a href="${result.chat_url}" target="_blank">打开官网查看 AI 回复</a>
                `;
                
                // 5秒后开始同步
                setTimeout(() => sync(result.session_id), 5000);
            }
        }
        
        async function sync(sessionId) {
            const response = await fetch(
                `${API_URL}/api/session/${sessionId}/messages`,
                { headers: { 'X-Bazi-Cookie': COOKIE } }
            );
            
            const result = await response.json();
            
            if (result.success) {
                const messages = result.messages;
                let html = '<h3>消息列表：</h3>';
                
                messages.forEach(msg => {
                    const role = msg.role === 'user' ? '👤' : '🤖';
                    html += `<p>${role} ${msg.content}</p>`;
                });
                
                document.getElementById('result').innerHTML = html;
                
                // 如果没有 AI 回复，继续同步
                const hasAI = messages.some(m => m.role === 'assistant');
                if (!hasAI) {
                    setTimeout(() => sync(sessionId), 5000);
                }
            }
        }
    </script>
</body>
</html>
```

## 📊 API 端点

### 基础端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | API 信息 |
| `/docs` | GET | API 文档 |
| `/health` | GET | 健康检查 |

### 功能端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/session/create` | POST | 创建会话 |
| `/api/session/<id>/send` | POST | 发送消息 |
| `/api/session/<id>/messages` | GET | 获取消息 |
| `/api/chat` | POST | 一键对话（推荐） |

**所有功能端点都需要在请求头中提供：**

```
X-Bazi-Cookie: 你的Cookie
```
