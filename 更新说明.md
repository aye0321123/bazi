## ✨ 新增功能

### 1. 创建新对话功能

现在你可以直接在天机阁中创建新的聊天会话！

#### 使用方法：

**在网页界面中：**
1. 登录天机阁（只需要 Cookie，不需要 Session ID）
2. 点击 **"✨ 新对话"** 按钮
3. 输入第一条消息（可选）
4. 系统自动创建新会话
5. 可以选择在新标签页中打开

**通过 API 调用：**

```python
import requests

# 方式 1: 只创建新会话
response = requests.post('http://127.0.0.1:5000/api/create-session')
result = response.json()
print(f"新会话 ID: {result['session_id']}")

# 方式 2: 创建新会话并发送消息
response = requests.post(
    'http://127.0.0.1:5000/api/create-and-send',
    json={'content': '你好，请用一句话介绍你自己'}
)
result = response.json()
print(f"对话链接: {result['chat_url']}")
```

## 🎯 使用场景

### 场景 1: 快速创建新对话

```
用户操作：

2. 登录（只需要 Cookie）
3. 点击"✨ 新对话"
4. 输入"请帮我分析一下今天的运势"
5. 点击"在新标签页中打开"

结果：
- 自动创建新会话
- 消息已发送
- 在 BaziAI 官网查看 AI 回复
```

### 场景 2: 批量创建对话（API）

```python
import requests
import json

# 读取 Cookie
with open('bazi_credentials.json', 'r') as f:
    cookie = json.load(f)['cookie']

# 登录
session = requests.Session()
session.post('http://127.0.0.1:5000/api/login', 
             json={'session_id': 'temp', 'cookie': cookie})

# 批量创建对话
questions = [
    "请帮我分析一下今天的运势",
    "我的事业运如何",
    "我的财运怎么样"
]

for question in questions:
    response = session.post(
        'http://127.0.0.1:5000/api/create-and-send',
        json={'content': question}
    )
    result = response.json()
    print(f"对话链接: {result['chat_url']}")
```

## ⚠️ 注意事项

### 1. Cookie 要求
- 创建新会话只需要 Cookie
- 不需要提供 Session ID
- Cookie 必须有效（未过期）

### 2. AI 回复机制
- 创建新会话后，消息已发送
- **AI 回复需要在 BaziAI 官网上查看**
- 等待 5-10 秒后，回到天机阁点击"刷新"

### 3. 会话管理
- 每次创建新会话会生成新的 Session ID
- 旧的 Session ID 仍然有效
- 可以通过重新登录切换会话

### 4. 浏览器兼容性
- 支持所有现代浏览器
- 需要启用 JavaScript
- 需要允许弹出窗口（用于打开新标签页）

## 🐛 已知问题

### 1. AI 回复延迟
- **问题**：发送消息后，AI 不会立即回复
- **原因**：BaziAI 使用 WebSocket/SSE 触发回复
- **解决**：在官网上等待回复，然后在天机阁中刷新

### 2. Cookie 过期
- **问题**：Cookie 过期后无法创建新会话
- **解决**：运行 `python get_cookie_v2.py` 重新获取

### 3. 跨域问题
- **问题**：某些浏览器可能阻止跨域请求
- **解决**：使用 Chrome 或 Firefox，或配置 CORS
