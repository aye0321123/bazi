## 🚀 方案 1: Railway（最推荐）

### 为什么选择 Railway？
- ✅ 完全免费（$5 免费额度/月）
- ✅ 部署超级简单（3 分钟完成）
- ✅ 自动 HTTPS
- ✅ 自动重启
- ✅ 提供免费域名

### 部署步骤

#### 1. 准备 GitHub 仓库

```bash
# 初始化 Git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "准备部署到 Railway"

# 创建 GitHub 仓库（在 GitHub 网站上创建）
# 然后推送
git remote add origin https://github.com/你的用户名/bazi-ai.git
git branch -M main
git push -u origin main
```

#### 2. 部署到 Railway

1. 访问 [https://railway.app](https://railway.app)
2. 点击 **"Start a New Project"**
3. 选择 **"Deploy from GitHub repo"**
4. 授权 GitHub 并选择你的仓库
5. Railway 会自动检测并部署

#### 3. 等待部署完成

- 部署通常需要 2-3 分钟
- 完成后会显示一个 URL，例如：`https://bazi-ai-production.up.railway.app`

#### 4. 访问你的应用

点击 Railway 提供的 URL，就可以访问你的天机阁了！

### 配置自定义域名（可选）

1. 在 Railway 项目设置中
2. 点击 **"Settings"** → **"Domains"**
3. 添加你的域名
4. 在域名服务商处添加 CNAME 记录

## 🎨 方案 2: Render（免费方案）

### 部署步骤

#### 1. 注册 Render

访问 [https://render.com](https://render.com) 并注册

#### 2. 创建 Web Service

1. 点击 **"New +"** → **"Web Service"**
2. 连接你的 GitHub 仓库
3. 配置：
   - **Name**: `bazi-ai`
   - **Environment**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn app:app`
   - **Instance Type**: `Free`

#### 3. 部署

点击 **"Create Web Service"**，Render 会自动部署

#### 4. 访问

部署完成后，Render 会提供一个 URL：`https://bazi-ai.onrender.com`

### ⚠️ 注意事项

- 免费版会在 15 分钟无活动后休眠
- 首次访问可能需要等待 30 秒唤醒

## 🐳 方案 3: Docker 部署（适合有服务器的用户）

### 前提条件

- 有一台云服务器（阿里云、腾讯云、AWS 等）
- 已安装 Docker

### 快速部署

#### 1. 连接到服务器

```bash
ssh root@你的服务器IP
```

#### 2. 克隆项目

```bash
git clone https://github.com/你的用户名/bazi-ai.git
cd bazi-ai
```

#### 3. 使用 Docker Compose 部署

```bash
# 启动
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止
docker-compose down
```

#### 4. 配置 Nginx（可选，用于域名访问）

创建 Nginx 配置文件：

```bash
nano /etc/nginx/sites-available/bazi-ai
```

内容：

```nginx
server {
    listen 80;
    server_name 你的域名.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

启用配置：

```bash
ln -s /etc/nginx/sites-available/bazi-ai /etc/nginx/sites-enabled/
nginx -t
systemctl reload nginx
```

#### 5. 配置 SSL（推荐）

```bash
# 安装 Certbot
apt install certbot python3-certbot-nginx -y

# 获取证书
certbot --nginx -d 你的域名.com

# 自动续期
certbot renew --dry-run
```

## 🇨🇳 方案 4: 阿里云/腾讯云（国内访问快）

### 步骤 1: 购买云服务器

#### 阿里云 ECS
- 访问 [https://www.aliyun.com](https://www.aliyun.com)
- 选择 **"云服务器 ECS"**
- 配置：1核2G，Ubuntu 20.04
- 学生价：约 ¥10/月

#### 腾讯云 CVM
- 访问 [https://cloud.tencent.com](https://cloud.tencent.com)
- 选择 **"云服务器"**
- 配置：1核2G，Ubuntu 20.04
- 学生价：约 ¥10/月

### 步骤 2: 连接服务器

```bash
ssh root@你的服务器IP
```

### 步骤 3: 安装依赖

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Python 和 Git
apt install python3 python3-pip git -y

# 安装 Docker（可选）
curl -fsSL https://get.docker.com | sh
```

### 步骤 4: 部署应用

#### 方式 A: 直接运行

```bash
# 克隆项目
git clone https://github.com/你的用户名/bazi-ai.git
cd bazi-ai

# 安装依赖
pip3 install -r requirements.txt

# 运行应用
gunicorn --bind 0.0.0.0:5000 --workers 4 app:app
```

#### 方式 B: 使用 Docker

```bash
# 克隆项目
git clone https://github.com/你的用户名/bazi-ai.git
cd bazi-ai

# 启动
docker-compose up -d
```

### 步骤 5: 配置开机自启

创建 systemd 服务：

```bash
nano /etc/systemd/system/bazi-ai.service
```

内容：

```ini
[Unit]
Description=BaziAI Web Application
After=network.target

[Service]
User=root
WorkingDirectory=/root/bazi-ai
ExecStart=/usr/local/bin/gunicorn --bind 0.0.0.0:5000 --workers 4 app:app
Restart=always

[Install]
WantedBy=multi-user.target
```

启动服务：

```bash
systemctl daemon-reload
systemctl start bazi-ai
systemctl enable bazi-ai
systemctl status bazi-ai
```

### 步骤 6: 配置防火墙

```bash
# 开放 80 和 443 端口
ufw allow 80
ufw allow 443
ufw enable
```

## 🔐 安全配置

### 1. 设置强密钥

```bash
# 生成随机密钥
python -c "import secrets; print(secrets.token_hex(32))"
```

将生成的密钥设置为环境变量：

**Railway/Render:**
在项目设置中添加环境变量 `SECRET_KEY`

**Docker:**
修改 `docker-compose.yml` 中的 `SECRET_KEY`

**服务器:**
```bash
export SECRET_KEY="你生成的密钥"
```

### 2. 限制访问（可选）

如果只想让特定人访问，可以添加 IP 白名单：

修改 `app.py`，在开头添加：

```python
ALLOWED_IPS = ['123.456.789.0', '111.222.333.444']

@app.before_request
def limit_remote_addr():
    if request.remote_addr not in ALLOWED_IPS:
        abort(403)
```

## 🌐 绑定域名

### 1. 购买域名

推荐：
- 阿里云：[https://wanwang.aliyun.com](https://wanwang.aliyun.com)

- GoDaddy：[https://www.godaddy.com](https://www.godaddy.com)

### 2. 配置 DNS

#### Railway/Render
添加 CNAME 记录：
```
类型: CNAME
主机记录: @
记录值: 平台提供的域名
```

#### 自己的服务器
添加 A 记录：
```
类型: A
主机记录: @
记录值: 你的服务器IP
TTL: 600
```

通常需要 10 分钟到 24 小时

## 🆘 常见问题

### Q: 部署后无法访问？
A: 
1. 检查防火墙是否开放端口
2. 检查应用是否正常运行
3. 查看日志文件

### Q: Railway 免费额度用完了怎么办？
A: 
1. 升级到付费计划（$5/月）
2. 或切换到 Render 免费版
3. 或部署到自己的服务器

### Q: 如何更新应用？
A:
```bash
git pull
pip install -r requirements.txt --upgrade
systemctl restart bazi-ai  # 如果使用 systemd
docker-compose restart     # 如果使用 Docker
```

### Q: 如何备份数据？
A: 应用使用 session 存储，无需备份。如需持久化，可以添加数据库。
